## 📊 Почему 52 MB?

Размер программы 52 MB из-за включенных компонентов:

| Компонент | Размер | Назначение |
|-----------|--------|------------|
| **Python 3.10+** | ~25 MB | Встроенный интерпретатор |
| **PyQt5** | ~15 MB | Графический интерфейс |
| **Pillow (PIL)** | ~8 MB | Обработка изображений и EXIF |
| **Иконки и ресурсы** | ~2 MB | Визуальные элементы |

**Итого:** 52 MB - все необходимое в одном файле!

## ✨ Возможности

### 📸 Для фотографий
- **EXIF данные** - автоматическое именование по дате, камере, параметрам съемки
- **Умные шаблоны** - `{date}_{camera}_{iso}` → `2024-01-15_Canon_ISO100.jpg`
- **Пакетная обработка** - тысячи файлов за один клик

### 📄 Для документов
- **Нумерация** - `001`, `002`, `003` с настраиваемым форматом
- **Префиксы/суффиксы** - группировка файлов по проектам
- **Сортировка** - по имени, дате, размеру

### 🔧 Для продвинутых
- **Регулярные выражения** - сложные паттерны замены
- **Фильтрация** - по расширениям и размеру файлов
- **Безопасность** - резервные копии и откат операций

## 📋 Системные требования

- **ОС:** Windows 10/11 (64-bit)
- **ОЗУ:** 4 GB (рекомендуется 8 GB)
- **Место на диске:** 100 MB свободного (52 MB программа + место для файлов)
- **Разрешение:** 1024×768 или выше

## 🛠️ Как использовать

### Базовый workflow:
1. **Выберите папку** с файлами
2. **Загрузите файлы** в таблицу
3. **Настройте правила** во вкладках
4. **Проверьте предпросмотр**
5. **Примените изменения**

### EXIF шаблоны:
```text
{date}        → 2024-01-15
{time}        → 14-30
{camera}      → Canon EOS 5D
{focal}       → 50mm
{aperture}    → F2.8
{iso}         → ISO100
```

**Полный шаблон:** `{date}_{camera}_{focal}mm_F{aperture}_ISO{iso}`

## ⚠️ Важная информация

### 🔒 Безопасность
- ✅ Не требует прав администратора
- ✅ Создает резервные копии
- ✅ Полный откат операций
- ✅ Локальная обработка (нет отправки данных)
- ✅ Проверено VirusTotal (результаты ниже)

### 📝 Ограничения
- EXIF работает только с изображениями
- Максимальная длина имени: 255 символов
- Запрещенные символы: `<>:"/\|?*`

### 🖼️ Поддерживаемые форматы
| Тип | Форматы |
|-----|---------|
| Изображения | JPG, PNG, TIFF, BMP, GIF, WebP, HEIC |
| RAW фото | NEF, CR2, ARW, DNG |
| Документы | PDF, DOCX, XLSX, TXT (без EXIF) |

## 🆘 Помощь и поддержка

### Частые проблемы:
<details>
<summary>📌 Программа не запускается (52 MB скачано)</summary>

1. **Проверьте антивирус** - добавьте исключение
2. **Место на диске** - нужно минимум 100 MB свободно
3. **Версия Windows** - только Windows 10/11 64-bit
4. **Запуск от администратора** - попробуйте правой кнопкой → "Запуск от имени администратора"
</details>

<details>
<summary>📌 EXIF не работает</summary>

1. Проверьте, что файл - изображение
2. Убедитесь в наличии EXIF данных
3. Попробуйте другой файл
4. Проверьте расширение файла
</details>

<details>
<summary>📌 Ошибка памяти при работе с большими файлами</summary>

1. Закройте другие программы
2. Уменьшите количество одновременно обрабатываемых файлов
3. Проверьте свободную оперативную память
</details>
